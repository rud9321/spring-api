# Spring Boot API with mongodb

![Build Status](https://travis-ci.org/joemccann/dillinger.svg?branch=master)

###### Default port is 7892 [RUN](http://127.0.0.1:7892/swagger-ui.html)

Spring Boot mongodb enabled api ready architecture to use

- Downlod
- Update as per your requrement

```sh
$ mvn install
$ mvn spring-boot:run
```

For production environments...
After running application DbSeeder class can be deleted.

### Dependencies

Swagger Dependencies

```sh
        <dependency>
			<groupId>io.springfox</groupId>
			<artifactId>springfox-swagger2</artifactId>
			<version>2.6.1</version>
			<scope>compile</scope>
		</dependency>
		<dependency>
			<groupId>io.springfox</groupId>
			<artifactId>springfox-swagger-ui</artifactId>
			<version>2.6.1</version>
			<scope>compile</scope>
		</dependency>
```
